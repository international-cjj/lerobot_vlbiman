# CjjArm 真机与 Canonical MuJoCo 仿真使用说明

本文档只保留当前有效的运行方式。`cjjarm_sim` 现在默认使用统一的 canonical 仿真链路，不再以 raw URDF 作为默认入口。

## 1. 当前默认仿真架构

默认链路：

```text
TRLC-DK1-Follower-home.mjcf
  -> build_dual_camera_scene(...)
  -> generated_dual_camera_target_scene.mjcf
  -> MjModel.from_xml_string(...)
```

默认仿真语义：

- 机械臂使用带 `home keyframe`、`position actuator`、`damping/armature` 的 MJCF。
- `step/action` 走 `data.ctrl`，不再在控制步骤中直接写 `data.qpos`。
- 默认相机固定为：
  - `front_camera` -> `observation.images.front`
  - `wrist_camera` -> `observation.images.wrist`
- 默认场景固定包含：
  - `floor_geom`
  - `table_geom`
  - `target_cube` + `target_cube_geom` + `freejoint`
- 默认夹爪碰撞使用简化 box collision 和内侧 pad：
  - `finger_left_inner_pad`
  - `finger_right_inner_pad`
- `virtual_target_body` 仍会存在，但它现在只是辅助参考目标，不是默认的物理抓取测试物。

兼容说明：

- `--legacy-scene` 仍然保留，但只用于兼容调试。
- raw URDF 模式默认禁用，不再作为常规使用路径。

## 4. 统一场景导出与查看

### 4.1 导出统一双相机场景

```bash
PYTHONPATH=src:. "$PY" \
  src/lerobot/projects/vlbiman_sa/app/run_mujoco_dual_camera_scene.py \
  --run-id latest_scene \
  --settle-steps 300 \
  --render-width 640 \
  --render-height 480
```

输出目录默认在：

[`outputs/vlbiman_sa/mujoco_dual_camera_scene/`](./outputs/vlbiman_sa/mujoco_dual_camera_scene/)

重点文件：

- `scene/dual_camera_target_scene.mjcf`
- `front_camera.png`
- `wrist_camera.png`
- `overview.png`
- `summary.json`

### 4.2 交互查看统一场景

```bash
PYTHONPATH=src:. "$PY" \
  src/lerobot/projects/vlbiman_sa/app/run_mujoco_dual_camera_viewer.py \
  --run-id viewer_latest \
  --settle-steps 300
```

按键：

- `1`: `front_camera`
- `2`: `wrist_camera`
- `3`: overview
- `Esc`: 退出

如果只写场景不打开 GUI：

```bash
PYTHONPATH=src:. "$PY" \
  src/lerobot/projects/vlbiman_sa/app/run_mujoco_dual_camera_viewer.py \
  --run-id viewer_scene_only \
  --write-scene-only
```

## 6. 数据录制

### 6.1 仿真录制

```bash
PYTHONPATH=src:. "$PY" -m lerobot.scripts.lerobot_record \
  --robot.type=cjjarm_sim \
  --robot.use_viewer=true \
  --teleop.type=keyboard_pose \
  --dataset.repo_id=local/cjjarm_sim_test \
  --dataset.root=outputs_cjjarm_sim_test \
  --dataset.single_task="pure sim manual demo" \
  --dataset.manual_episode_control=true \
  --dataset.num_episodes=1 \
  --dataset.episode_time_s=0 \
  --dataset.reset_time_s=0 \
  --dataset.fps=10 \
  --dataset.video=true \
  --dataset.push_to_hub=false
```

`cjjarm_sim` 在这里默认就是 canonical actuator-driven MuJoCo 仿真。

录制时：

- `Space`：开始 / 结束当前 episode
- `Enter`：保存当前 episode
- `Backspace`：丢弃当前 episode
- `Esc`：退出录制
- `Left Shift / Right Shift`：`Z+ / Z-`

### 6.2 真机录制

```bash
PYTHONPATH=src:. "$PY" -m lerobot.scripts.lerobot_record \

  --dataset.episode_time_s=3 \
  --dataset.reset_time_s=0 \
  --dataset.fps=10 \

  --dataset.push_to_hub=false
```

- `lerobot.projects.vlbiman_sa.sim.cjjarm_mujoco_env.CjjArmMujocoEnv`

## 10. 当前推荐使用顺序

1. 先跑 `tools/test_actuators.py`
2. 再跑 `test_cjjarm_sim_*` 和 `test_cjjarm_mujoco_env.py`
3. 再用 `run_mujoco_dual_camera_viewer.py` 看场景
4. 再用 `teleop_cjjarm_keyboard.py --robot-type cjjarm_sim` 做交互检查
5. 最后再进入录制、回放或训练
